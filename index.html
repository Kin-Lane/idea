API Ideas

This is a working project for my API Ideas API (deep), allowing the adding and listing of ideas for new APIs from existing sources (aka. Shadow API). You can GET ideas for APIs using the API documented below, if you want to submit an idea you will need to register for an API key.


If you have an idea for an API you'd like to see on this list, sign up for an API key, and POST your idea.

This API is maintained by Kin Lane (@kinlane), the API Evangelist. If you have any questions, comments or there are APIs you'd like to see deployed, you can submit them on this projects, Github repo issue management. I use 3Scale API Infrastructure to manage access to my APIs, across all parts of the API Evangelist network.
